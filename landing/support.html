Questions fréquentes

💜 Comment fonctionne le score de compatibilité ?

NextLove analyse 7 dimensions psychologiques (valeurs, famille, mode de vie, intimité, loisirs, ambitions, personnalité) à partir de tes réponses au questionnaire. Le score est calculé par comparaison directe avec chaque profil.

💳 Comment annuler mon abonnement Gold ?

Sur iOS : Réglages iPhone → Ton identifiant Apple → Abonnements → NextLove → Annuler.
Sur Android : Play Store → Profil → Paiements → Abonnements → NextLove → Annuler.

🔒 Comment supprimer mon compte ?

Dans l'app : Profil → Paramètres → Supprimer le compte. Toutes tes données sont effacées sous 72h. Tu peux aussi envoyer un email à contact@nextlove.app.

👁️ Pourquoi je vois des profils sans photo ?

Certains utilisateurs n'ont pas encore ajouté de photo. Tu peux filtrer pour ne voir que les profils avec photo dans les filtres de découverte.

🚩 Comment signaler un profil ?

Ouvre la conversation ou le profil → appuie sur les 3 points (⋯) → Signaler. Notre équipe examine chaque signalement sous 24h.

🎁 J'ai une question sur l'offre "500 premiers pré-inscrits"

Les 500 premiers utilisateurs à créer un compte reçoivent 3 mois de Gold offerts automatiquement après validation de leur profil. Contacte-nous si tu as un problème d'activation.
